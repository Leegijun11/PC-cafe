## 🎮 PC-Cafe Kiosk Project
---
### 프로젝트 소개

PC방 키오스크를 웹으로 구현한 미니 프로젝트입니다.
HTML, CSS, JavaScript를 활용하여

메뉴 선택 → 장바구니 → 결제 방식 선택 → 결제 완료

까지의 흐름을 구현했습니다.

---

## 사용 기술

- HTML

- CSS

- JavaScript

- window.open() 을 이용한 결제창 분리

- dataset 속성을 이용한 가격 관리

- DOM 조작 및 이벤트 처리

---


## 프로젝트 구조
```
PC-cafe/ 
│   
├── PC-cafe.html        # 메인 키오스크 화면
├── receipt1.html       # 카드 결제 페이지
├── receipt2.html       # 현금 결제 페이지
│    
└── menu/               # 메뉴 이미지 폴더
    ├── ramen1.jpg 
    ├── side1.jpg
    ├── dinner1.jpg
    ├── drink1.jpg
    └── ...
```

※ menu 폴더 안에 이미지 파일들을 모두 넣어주세요.

## 주요 기능
### 1. 메뉴 카테고리 선택

- 라면류

- 사이드류

- 식사류

- 음료류

- 버튼 클릭 시 해당 메뉴만 표시됩니다.

---

### 2. 메뉴 선택 기능

- 메뉴 이미지를 클릭하면 confirm 창이 나타납니다.

- 확인을 누르면 장바구니에 추가됩니다.

- data-value 속성을 이용해 가격을 추출합니다.

- 총 금액이 자동 계산됩니다.

---

### 3. 결제 방식 선택

- 카드 결제

- 현금 결제

- 체크박스를 통해 하나의 결제 방식만 선택하도록 구현했습니다.

---

### 4. 카드 결제 페이지 (receipt1.html)

- 카드사 선택

- 전체 약관 동의 기능

- 약관 동의 후 결제 가능

- 결제 완료 2초 후 알림

---

### 5. 현금 결제 페이지 (receipt2.html)

- 현금결제 / 계좌이체 선택

- 약관 동의

- 현금영수증 발행 여부 체크

- 결제 완료 알림
